Adds the ability to do bulk attachments and re-attachments in the Media Library to any media object.

== Description ==

Simple plugin to enhance your ability to attach images to posts. 

This will allow you to more easily manage and use the WordPress native gallery shortcode and all associated plugins.

== Installation ==

1. Upload `media-library-plus.zip` onto your local computer.
2. Go to your WordPress Dashboard and select <strong>Plugins >> Add New</strong>.
3. Click on the <strong>Upload</strong> option at the top and select the `media-library-plus.zip file` you just downloaded.
4. Click on <strong>Install</strong>.
5. Activate the plugin through the 'Plugins' menu in WordPress
6. There should be an additional drop down menu in the WordPress Media Library that allows you to do bulk image attachments to any media object.

== Frequently Asked Questions ==

= Are you going to add more functionality? =

I am currently working on adding the ability to create light-weight image grouping objects. This will enable simple and quick creation of image groups that can then be displayed using the WordPress native gallery shortcode.
